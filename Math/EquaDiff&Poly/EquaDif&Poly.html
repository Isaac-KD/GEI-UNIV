Équations Différentielles Linéaires d'Ordre 1

∫

Le jury ne veut pas voir une page de calcul. Il te donne l'équation et 5 solutions. Ton but est de reconnaître la forme générale ou d'utiliser la méthode inversée : tester les solutions proposées directement.

La solution générale est toujours : \(y = y_h + y_p\) — solution homogène + une solution particulière évidente.

Forme générale & solution homogène
Forme générale
\(y' + a(x)\,y = b(x)\)
Solution homogène — équation sans second membre (\(b = 0\))
\(y_h(x) = C \cdot \exp\!\left(-\int a(x)\,dx\right)\)
⚠ Attention au signe "-" dans l'exposant — piège classique. Cas le plus fréquent (\(a\) = constante) : \(y_h(x) = C \cdot e^{-ax}\).
Solution particulière \(y_p\)
On cherche une solution particulière \(y_p\) qui vérifie l'équation complète avec le second membre \(b(x)\). En QCM, ne pas chercher avec la méthode de variation de la constante — utiliser directement le hack ci-dessous.
⚡ Hack Sniper — Méthode QCM infaillible

Prends les 5 formules proposées (A, B, C, D, E), dérive-les mentalement et regarde laquelle vérifie \(y' + ay = b(x)\). C'est mathématiquement infaillible et 3× plus rapide que de construire \(y_p\) de zéro.

En pratique : pour \(y' + 2y = 4\), la solution particulière est \(y_p = 2\) (constante). Vérifie : \(0 + 2 \times 2 = 4\) ✓. Si une réponse ne vérifie pas l'équation, elle est éliminée.

Équations Différentielles d'Ordre 2 — Coefficients Constants

〰️

Les 3 cas de l'équation caractéristique sont cruciaux en Maths ET en Physique (circuits RLC, oscillateurs mécaniques). La même structure apparaît partout.

Méthode — Équation caractéristique
Forme générale homogène
\(ay'' + by' + cy = 0\)
Étape 1 — Poser l'équation caractéristique
\(ar^2 + br + c = 0 \qquad \Delta = b^2 - 4ac\)
On substitue \(y = e^{rx}\) dans l'équation. Les solutions de cet polynôme donnent les exposants des solutions.
Étape 2 — Analyser Δ et écrire la solution
Δ > 0
Régime apériodique
Deux racines réelles distinctes : \(r_1\) et \(r_2\)
\(y(x) = A\,e^{r_1 x} + B\,e^{r_2 x}\)
Δ = 0
Régime critique
Une racine double : \(r_0 = -\dfrac{b}{2a}\)
\(y(x) = (Ax + B)\,e^{r_0 x}\)
⚠ Piège : ne pas oublier le "\(x\)" devant le \(A\) !
Δ < 0
Régime pseudo-périodique / Oscillations
Deux racines complexes conjuguées : \(r = \alpha \pm i\beta\)
avec \(\alpha = -\dfrac{b}{2a}\) et \(\beta = \dfrac{\sqrt{-\Delta}}{2a}\)
\(y(x) = e^{\alpha x}\!\left[A\cos(\beta x) + B\sin(\beta x)\right]\)
⚡ Hack Conditions Initiales — Éliminer sans calculer A et B

Si on te donne des conditions initiales (ex : \(y(0) = 0\) ou \(y'(0) = 1\)), injecte directement \(x = 0\) dans les propositions du QCM.

Exemple : \(y(0) = 0\) élimine immédiatement toute solution dont la valeur en 0 est non nulle — sans avoir à calculer A et B.

Lien avec la physique : Ces 3 cas correspondent exactement aux régimes des oscillateurs (circuit RLC, ressort-masse-amortisseur). Δ > 0 = sur-amorti, Δ = 0 = critique, Δ < 0 = sous-amorti (oscillations).

Racines & Multiplicité

🎯

Si \(\alpha\) est racine d'un polynôme \(P\), alors \(P\) est factorisable par \((X - \alpha)\). La multiplicité d'une racine se lit sur ses dérivées successives.

Théorème de Multiplicité — Ultra-testé
Racine simple
\(P(\alpha) = 0\)  ET  \(P'(\alpha) \neq 0\)  →  Factorisable par \((X-\alpha)\)
Racine double
\(P(\alpha) = 0\)  ET  \(P'(\alpha) = 0\)  ET  \(P''(\alpha) \neq 0\)  →  Factorisable par \((X-\alpha)^2\)
Racine d'ordre k
\(P(\alpha) = P'(\alpha) = \cdots = P^{(k-1)}(\alpha) = 0\)  ET  \(P^{(k)}(\alpha) \neq 0\)  →  Factorisable par \((X-\alpha)^k\)
⚡ Hack — Trouver les racines d'un polynôme de degré 3 ou 4

Face à un polynôme difficile à factoriser :

  1. Calcule \(P'(X)\) — c'est de degré inférieur, donc plus facile.
  2. Trouve les racines de \(P'(X)\).
  3. Si une racine \(\alpha\) de \(P'\) vérifie aussi \(P(\alpha) = 0\) → bingo, c'est une racine double de \(P\) !

Cette méthode évite de chercher les racines à l'aveugle par essais et erreurs.

★ Exemple guidé

Soit \(P(X) = X^3 - 3X + 2\). On calcule \(P'(X) = 3X^2 - 3 = 3(X-1)(X+1)\). Racines de \(P'\) : \(X = 1\) et \(X = -1\).

On vérifie : \(P(1) = 1 - 3 + 2 = 0\) ✓. Donc \(X = 1\) est une racine double de \(P\). On peut écrire \(P(X) = (X-1)^2(X+2)\).

Formules de Viète — Relations Coefficients-Racines

🔗

Les formules de Viète permettent de lier les racines aux coefficients sans avoir à les calculer explicitement — économie de temps précieuse en QCM.

Cas général — Polynôme de degré n
Polynôme
\(P(X) = a_n X^n + a_{n-1} X^{n-1} + \cdots + a_1 X + a_0\)
Somme de toutes les racines
\(\displaystyle\sum x_i = -\frac{a_{n-1}}{a_n}\)
Rapport du coeff de X^(n-1) sur celui de X^n, avec signe opposé.
Produit de toutes les racines
\(\displaystyle\prod x_i = (-1)^n \frac{a_0}{a_n}\)
Rapport du terme constant sur le coeff dominant, avec signe \((-1)^n\).
Cas fondamental — Degré 2 : \(aX^2 + bX + c\)
Somme\(x_1 + x_2 = -\dfrac{b}{a}\) Produit\(x_1 \cdot x_2 = \dfrac{c}{a}\)
Application directe : Trouver deux nombres dont on connaît la somme \(S\) et le produit \(P\) revient à résoudre \(X^2 - SX + P = 0\).

Arithmétique des Polynômes — PGCD & Bézout

⚗️

Tout comme avec les entiers, on peut effectuer des divisions euclidiennes de polynômes : \(A = B \cdot Q + R\) avec \(\deg(R) < \deg(B)\). L'algorithme d'Euclide s'applique exactement de la même façon.

Division euclidienne de polynômes
Formule
\(A = B \cdot Q + R \quad \text{avec } \deg(R) < \deg(B)\)
\(Q\) est le quotient et \(R\) le reste. Si \(R = 0\), alors \(B\) divise \(A\).
Algorithme d'Euclide pour PGCD(A, B)
1
Divise le grand polynôme \(A\) par le petit \(B\). Obtiens un reste \(R_1\) (\(\deg(R_1) < \deg(B)\)).
2
Divise \(B\) par \(R_1\). Obtiens un reste \(R_2\).
3
Continue jusqu'à obtenir un reste nul.
★
Le PGCD = dernier reste non nul, rendu unitaire (divisé par son coefficient dominant pour qu'il commence par \(1 \cdot X^k\)).
★ Théorème de Bézout pour les polynômes

Deux polynômes sont premiers entre eux si leur PGCD est égal à 1 (polynôme constant).

Dans ce cas, le théorème de Bézout s'applique : il existe des polynômes \(U\) et \(V\) tels que :

\[A \cdot U + B \cdot V = 1\]

Application : si \(\text{PGCD}(A, B) = D\), on peut toujours écrire \(A \cdot U + B \cdot V = D\) pour certains polynômes \(U\) et \(V\).
